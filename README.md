# calendar-react

**Веб-календарь на React**  
Интерактивное приложение-календарь, переписанно с репозитория https://github.com/Torens29/calendar с использованием библиотеки React.

## О проекте
Проект представляет собой классический учебный пример React-приложения.  
На основе структуры и коммитов можно предположить следующий функционал:
- Отображение сетки календаря (месяц/неделя).
- Переключение между датами.
- Добавление/отображение событий или заметок (опционально).
- Адаптивная вёрстка и стилизация.


## Технологии
- **React** — библиотека для построения пользовательских интерфейсов.
- **Create React App** — быстрая настройка сборки и разработки.
- **JavaScript (ES6+)** — логика компонентов.
- **CSS3** — стилизация.
- **HTML5** — базовая разметка.
